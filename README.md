# 🌦 ML Прогноз опадів

## Опис проєкту
Це Streamlit-застосунок для прогнозування опадів на основі історичних даних з сервісу [Open-Meteo](https://open-meteo.com/).  
Застосунок дозволяє:
- Завантажувати історичні дані для обраного міста
- Навчати модель RandomForestClassifier
- Прогнозувати на кілька наступних днів

## Дані
Дані отримуються через HTTP-запит до Open-Meteo (щоденні значення).  
Використані поля:
- `precipitation_sum` — сумарні опади за день
- `rain_sum` — сумарний дощ за день
- `temperature_2m_max` — максимальна температура
- `temperature_2m_min` — мінімальна температура
- `windspeed_10m_max` — максимальна швидкість вітру
- `rain_label` — цільова змінна (0 — без опадів, 1 — є опади)

CSV-файл з історичними даними: **`weather_daily.csv`**

## Використана модель
- **RandomForestClassifier** з бібліотеки scikit-learn  
- Цільова змінна (`rain_label`):
  - 0 — опадів немає (`precipitation_sum = 0`)
  - 1 — опади є (`precipitation_sum > 0`)
